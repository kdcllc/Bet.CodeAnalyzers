Re-usable 'ruleset' nuget package for Code Analyzers.

## Intall

```bash
     # analyzers and rules
     dotnet add package Bet.CodeAnalyzers

     # default .editorconfig
     dotnet add package Bet.EditorConfig
```


Cached packages location `%USERPROFILE%\.nuget\packages\`.
